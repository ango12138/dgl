# DGL - Knowledge Graph Embedding


## Models

DGL-KE now support knowledge graph embedding models including:

- [x] TransE
- [x] TransH
- [x] TransR
- [x] TransD
- [x] RESCAL
- [x] DistMult
- [x] ComplEx
- [x] RotatE

## Datasets

DGL-KE is tested on datasets including:

- [x] FB15k
- [x] FB15k-237
- [x] wn18
- [x] wn18rr
- [ ] Freebase
